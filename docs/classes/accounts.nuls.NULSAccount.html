Class NULSAccount

NULSAccount implements the Account class for the NULS protocol. It is used to represent a NULS account when publishing a message on the Aleph network.

Sign

  • Sign(message: BaseMessage): Promise<string>

  • Overrides Account.Sign

    • Defined in accounts/nuls.ts:64

    The Sign method provides a way to sign a given Aleph message using a NULS account. The full message is not used as the payload, only fields of the BaseMessage type are.

    The message's signature is based on secp256k1 package.

    Parameters

    • message: BaseMessage

      The Aleph message to sign, using some of its fields.
